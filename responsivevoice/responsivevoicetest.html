Here's a test of ResponsiveVoice.js library. Start by pressing the buttons under this text description.

Feel free to press F12 and play around with the code. You can also right click on this page and and select view page source (in Chrome) to look inside the code.

If you want to make the changes pernament:

  • press CTRL+S to save this webpage to HDD in your computer.
  • Next, open the source code using notepad++
  • Edit the .html and save it again somewhere
  • Open it using Notepad menu Run->Launch in Chrome. (or in Firefox, whichever you have and like).

There are buttons (input tag) inside html. they have onmouseover attribute that runs javascript. Like this: 

onclick=" responsiveVoice.cancel(); responsiveVoice.speak('Hello neighbor!');"

First command responsiveVoice.cancel stops any text speech that is in progress. Next, responsiveVoice.speak speaks a text that is inside the parenthesis and apostrophes. Use ordinary straight apostrophes ', not the back quote `.

Some of the javascript is inside functions. For example inside the rekt function. You can modify the function and the button named Test 2 will run your code.
